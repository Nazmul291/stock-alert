# Stock Alert — Feature Sheet

Real-time inventory monitoring & back-in-stock automation for Shopify

**Never miss a restock, a stockout, or a lost sale.**

| Category | What it does |
|---|---|
| **Live Monitoring** | Real-time dashboard, at-risk product alerts, webhook-powered sync (seconds, not hours) |
| **Predictive Stock-Outs** | "~3 days left" predictions from 30-day sales velocity, auto-calculated reorder-by dates |
| **Smart Automation** | Auto-hide sold-out products, auto-republish on restock (Pro) |
| **Multi-Channel Alerts** | Email, one-click Slack Connect (Pro), and outbound webhooks (Zapier/Make/ERP, Pro) — plus daily/weekly digest emails |
| **Automation & Marketing** | Native Shopify Flow triggers (low stock, out of stock, restock) and a Klaviyo integration — store alerts plus per-customer back-in-stock events for real marketing flows |
| **Back-in-Stock Widget** | No-code "Notify Me" form on the storefront, auto-emails customers the second stock returns |
| **Analytics** | 30-day trends, alert breakdowns, top at-risk products, stock health overview |
| **White-Label Emails** | Custom logo, brand color, sender name on all alert emails (Pro) |
| **Built for Scale** | Per-product thresholds, collection/tag filtering, bulk actions, CSV export |

**Pricing:** Basic $3.99/mo (1,000 products, email + Klaviyo + Shopify Flow) · Pro $9.99/mo (10,000 products, adds Slack Connect, outbound webhooks, branding) — **30-day free trial on both.**
